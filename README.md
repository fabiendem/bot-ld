## Getting Started

Install the module with: `npm install bot-ld`

```js
var bot-ld = require('bot-ld');
bot-ld.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g bot-ld
$ bot-ld --help
$ bot-ld --version
```


```sh
# creates a browser.js
$ grunt browserify
```
